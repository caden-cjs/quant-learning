## 📖 知识点

### MA 计算原理

```
MA5 = (Day1 + Day2 + Day3 + Day4 + Day5) / 5
MA10 = 最近10天收盘价之和 / 10
MA20 = 最近20天收盘价之和 / 20
```

### Pandas rolling() 函数

```python
df['MA5'] = df['close'].rolling(window=5).mean()
df['MA10'] = df['close'].rolling(window=10).mean()
df['MA20'] = df['close'].rolling(window=20).mean()
```

### 不同周期 MA 的意义

| MA周期 | 特点 | 用途 |
|--------|------|------|
| MA5 | 变化快，贴近当前价格 | 短期趋势判断 |
| MA10 | 中等平滑度 | 中短期交易参考 |
| MA20 | 变化慢，更平滑 | 长期趋势判断 |

### 金叉与死叉

- **金叉**：MA5 从下往上穿过 MA20 → 可能是买入信号 ✅
- **死叉**：MA5 从上往下穿过 MA20 → 可能是卖出信号 ❌
