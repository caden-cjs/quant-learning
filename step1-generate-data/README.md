## 📖 知识点

### 什么是 OHLCV？

股票数据通常包含以下字段：

| 字段 | 英文 | 含义 |
|------|------|------|
| 开盘价 | Open | 当天第一笔交易的价格 |
| 最高价 | High | 当天最高的成交价格 |
| 最低价 | Low | 当天最低的成交价格 |
| 收盘价 | Close | 当天最后一笔交易的价格 |
| 成交量 | Volume | 当天成交的股票数量 |

这是量化交易最基础的数据格式，几乎所有策略都从 OHLCV 开始！

### 关键概念

- **DataFrame**: Pandas 的核心数据结构，类似 Excel 表格
- **日期索引**: 量化分析的标准做法，方便时间序列处理
- **随机游走模型**: 用前一天价格 × (1 + 随机波动) 模拟价格变化
